# 🏗️ Construction Website

A modern and responsive construction company website designed to
showcase services, projects, and allow customers to contact directly
using a working contact form powered by Formspree.

## 🚀 Features

-   Responsive design
-   Service showcase
-   Contact form using Formspree
-   WhatsApp contact button
-   Google Maps integration
-   SEO-friendly structure

## 🛠️ Technologies Used

-   HTML5
-   CSS3
-   Formspree
-   GitHub Pages

## 📩 Contact Form Setup (Formspree)

1.  Go to https://formspree.io\
2.  Create an account\
3.  Create a new form\
4.  Copy your endpoint URL

Example: https://formspree.io/f/your-form-id
